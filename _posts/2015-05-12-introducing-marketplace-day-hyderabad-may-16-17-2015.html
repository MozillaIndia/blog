Ever wondered how someone contributes to an open source code-base ? Do you want to be one such contributor? Here is the opportunity – Marketplace Day (Hyderabad), an event completely dedicated to Mozilla’s Firefox Marketplace Code Contribution. If you are in and around Hyderabad, do join us for the event.

Firefox Marketplace is the only app store dedicated to providing content built with HTML5 standards and open source values. To help bring openness, freedom, and user choice to the world of app publishing, we invite you to join the awesome Firefox Marketplace community.

Marketplace Day is going to be completely dedicated to Firefox Marketplace code contribution. This is going to be an open event, and we are expecting to on-board many new contributors to Marketplace community.

You will learn

You will learn everything necessary for getting started with code-base contribution

  • Linux & terminal
  • Git & github
  • Bugzilla
  • irc
  • Web development best practices
  • Marketplace code-base go-through
  • Fixing bugs for Firefox marketplace
